#   (free, takes ~2 min, get a clientId GUID back)
CW_COMPANY_ID=bluefoxgroup
CW_PUBLIC_KEY=your-public-key
CW_PRIVATE_KEY=your-private-key
CW_CLIENT_ID=your-client-id-guid

# --- Server port (default 3978 is the Bot Framework convention) ---
PORT=3978


# ============================================================
# SETUP GUIDE (one-time, ~30-45 min total)
# ============================================================
#
# STEP 1 — ConnectWise API Key
#   1. CW Manage → System → Members → [your member] → API Keys tab
#   2. New Key → description "Teams Bot" → Save
#   3. Copy Public Key → CW_PUBLIC_KEY
#   4. Copy Private Key → CW_PRIVATE_KEY (shown once!)
#   5. Go to developer.connectwise.com → register app → get CW_CLIENT_ID
#
# STEP 2 — Azure App Registration
#   1. portal.azure.com → Azure Active Directory → App Registrations
#   2. New Registration → Name: "BFG CW Bot" → Single Tenant → Register
#   3. Copy Application (client) ID → TEAMS_APP_ID
#   4. Copy Directory (tenant) ID → TEAMS_TENANT_ID
#   5. Certificates & Secrets → New client secret → TEAMS_APP_PASSWORD
#
# STEP 3 — Azure Bot Resource
#   1. portal.azure.com → Create Resource → "Azure Bot"
#   2. Use existing app registration → paste your App ID
#   3. Channels → enable Microsoft Teams
#   4. Messaging Endpoint: https://YOUR-DOMAIN/api/messages
#      (use ngrok for local dev: ngrok http 3978 → use the https URL)
#
# STEP 4 — Teams App Package
#   1. Go to dev.teams.microsoft.com (Teams Developer Portal)
#   2. Apps → New App → fill in name/description
#   3. App Features → Bot → connect to your Azure Bot
#   4. Scopes: "Team" + "Group Chat"
#   5. Publish → Download app package (.zip)
#   6. In Teams: Apps → Manage apps → Upload custom app → upload the zip
#
# STEP 5 — Test it
#   npm install && npm start
#   ngrok http 3978 (paste the https URL into Azure Bot messaging endpoint)
#   In Teams, paste any ticket number like #357082 into a channel
